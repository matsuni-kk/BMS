# 離婚手続きの法的要件分析

## 1. 離婚の種類と要件
### 1.1 協議離婚
- 要件：双方の合意
- 必要書類：
  - 離婚届
  - 署名・捺印
  - 成年者の証人2名の署名・捺印
- 手続き場所：市区町村役場

### 1.2 調停離婚
- 申立要件：当事者の一方による申立て
- 管轄：家庭裁判所
- 必要書類：
  1. 調停申立書
  2. 戸籍謄本
  3. 附属書類

### 1.3 裁判離婚
- 要件：民法770条の離婚事由の存在
  1. 不貞行為
  2. 悪意の遺棄
  3. 生死不明（3年以上）
  4. 回復の見込みのない精神病
  5. その他婚姻を継続し難い重大な事由
- 証拠の必要性
