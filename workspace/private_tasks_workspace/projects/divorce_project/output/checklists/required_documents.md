# 協議離婚 必要書類・手続きチェックリスト

## 1. 事前準備段階

### 1.1 本人確認書類の準備
- [ ] 運転免許証
  - 有効期限を確認
  - 住所が現在のものと一致しているか確認
- [ ] マイナンバーカード
  - 有効期限を確認
  - 住所情報が最新であることを確認
- [ ] パスポート（国際結婚の場合）
  - 有効期限の確認
  - 必要に応じて翻訳証明書の準備

### 1.2 戸籍関係書類
- [ ] 戸籍謄本
  - 発行後3ヶ月以内のもの
  - 夫婦それぞれの本籍地の市区町村役場で取得
  - 必要部数：2通以上（離婚届提出用、その他手続き用）
- [ ] 戸籍の附票
  - 過去の住所履歴が記載されたもの
  - 本籍地の市区町村役場で取得

### 1.3 住民関係書類
- [ ] 住民票
  - 世帯全員のもの
  - マイナンバーの記載なしで可
  - 発行後3ヶ月以内のもの
- [ ] 印鑑証明書
  - 発行後3ヶ月以内のもの
  - 実印を使用する場合に必要

## 2. 財産分与関連書類

### 2.1 不動産関係
- [ ] 不動産登記簿謄本
  - 所有するすべての不動産について取得
  - 発行後3ヶ月以内のもの
- [ ] 固定資産評価証明書
  - 市区町村の税務課で取得
  - 不動産の評価額の証明として使用
- [ ] 住宅ローン残高証明書
  - 金融機関で取得
  - 直近の返済状況を含むもの

### 2.2 金融資産関係
- [ ] 預貯金通帳のコピー
  - すべての口座の写し
  - 結婚前からの資産も含める
  - 直近3年分程度
- [ ] 証券口座の取引履歴
  - 株式、投資信託等の評価額を示すもの
  - 直近の残高証明書
- [ ] 保険証券のコピー
  - 生命保険、損害保険等すべての保険証券
  - 解約返戻金額の確認資料

### 2.3 収入関係
- [ ] 源泉徴収票
  - 直近3年分
  - 夫婦双方のもの
- [ ] 確定申告書
  - 自営業の場合は直近3年分
  - 収支内訳書、決算書を含む
- [ ] 給与明細書
  - 直近1年分程度
  - 賞与明細書も含む

## 3. 親権・養育費関連（子どもがいる場合）

### 3.1 子どもの書類
- [ ] 戸籍謄本
  - 子どもが別戸籍の場合
  - 発行後3ヶ月以内のもの
- [ ] 住民票
  - 世帯全員のもの
  - 続柄の記載があるもの
- [ ] 健康保険証のコピー
  - 現在加入している保険の確認用

### 3.2 養育費関係
- [ ] 収支状況証明書類
  - 教育費の実績と予測
  - 医療費の実績
  - 習い事等の費用証明
- [ ] 養育費算定表
  - 裁判所の基準に基づく計算資料
  - 双方の収入を考慮した試算

## 4. 離婚届関連

### 4.1 必須書類
- [ ] 離婚届用紙
  - 市区町村役場またはウェブサイトから入手
  - 記入例を参照しながら正確に記入
- [ ] 夫婦それぞれの印鑑
  - シャチハタ等のゴム印は不可
  - 認印で可（実印である必要はない）

### 4.2 証人関係
- [ ] 証人2名の署名・押印
  - 成人であること
  - 夫婦の親族でも可
  - 証人の印鑑（認印可）

## 5. 離婚後の各種手続き

### 5.1 戸籍関係
- [ ] 戸籍謄本（離婚後）
  - 離婚届受理後に取得
  - 新しい戸籍の確認用
- [ ] 戸籍の附票（離婚後）
  - 住所履歴の更新確認用

### 5.2 住所・氏名変更関係
- [ ] 転出届・転入届（住所変更の場合）
  - 引越し先の市区町村役場に提出
  - マイナンバーカードの住所変更も忘れずに
- [ ] 氏の変更届（旧姓に戻る場合）
  - 離婚届と同時に提出可能
  - 3ヶ月以内の提出が望ましい

### 5.3 各種名義変更手続き
- [ ] 健康保険の切替手続き
  - 勤務先の健康保険または国民健康保険
  - 被扶養者から外れる手続き
- [ ] 年金手帳の名義変更
  - 年金事務所で手続き
  - 年金分割の手続きも検討
- [ ] 銀行口座の名義変更
  - 各金融機関での手続き
  - 通帳、カードの再発行
- [ ] クレジットカードの名義変更
  - 各カード会社での手続き
  - 必要に応じて解約・新規作成
- [ ] 携帯電話の名義変更
  - 契約者名義の変更
  - 家族割等のプラン変更
- [ ] 公共料金の名義変更
  - 電気、ガス、水道等
  - インターネット、ケーブルTV等

## 6. 子どもに関する手続き（該当する場合）

### 6.1 親権者変更関係
- [ ] 戸籍謄本（子ども）
  - 親権者変更の記録確認用
- [ ] 住民票の写し
  - 世帯構成の変更確認用

### 6.2 学校関係
- [ ] 学校への届出
  - 親権者変更の連絡
  - 緊急連絡先の変更
  - 授業料支払者の変更
- [ ] 保育園・幼稚園への届出
  - 家庭状況変更の連絡
  - 送迎者の変更手続き

## 7. その他必要に応じて準備する書類

### 7.1 財産分与の証明
- [ ] 財産分与契約書
  - 公正証書での作成を推奨
  - 弁護士等の専門家に相談
- [ ] 不動産の名義変更書類
  - 登記事項証明書
  - 登記簿謄本
  - 固定資産評価証明書

### 7.2 養育費関係
- [ ] 養育費契約書
  - 公正証書での作成を推奨
  - 支払い方法、期間の明記
- [ ] 養育費支払い口座の通帳コピー
  - 専用口座の開設
  - 通帳の写しを保管

## タスク実行上の注意点

1. 書類の有効期限
   - 公的書類は原則3ヶ月以内のものを使用
   - 期限切れに注意して余裕を持って取得

2. 必要部数
   - 原本が必要な手続きが複数ある場合は余分に取得
   - コピーで対応可能か事前に確認

3. 費用
   - 各種証明書の発行手数料を事前に確認
   - 複数部数の場合の追加料金も考慮

4. 手続きの順序
   - 離婚届提出前に必要な準備を完了
   - 離婚後の手続きは優先順位をつけて計画的に実行

5. 書類の保管
   - 重要書類は複数部コピーを作成
   - スキャンデータとしても保存を推奨
   - 安全な場所での保管を徹底

## 進捗管理

上記の各項目について、以下の状態を記録：
- [ ] 未着手
- [P] 準備中
- [C] 完了
- [N/A] 該当なし

定期的に進捗を確認し、必要に応じて優先順位の調整や追加のタスクを検討してください。
