# コミュニケーション記録

このディレクトリには、離婚協議に関連する過去のコミュニケーション記録を保管します。

## 保管対象資料
- メール履歴
- LINE等のメッセージログ
- 電話記録
- 面談記録
- 協議記録
- 調停記録（該当する場合）

## 記録の管理方法
### 分類基準
- 日付順に整理
- コミュニケーション手段ごとに分類
- 重要度によるラベリング

### 重要度の定義
- 高：法的な合意や重要な決定に関する内容
- 中：具体的な協議内容や提案
- 低：一般的な情報交換

## 記録方法
### 必須記載事項
- 日時
- 参加者
- 通信手段
- 主な内容
- 結論・合意事項（ある場合）
- フォローアップ事項

## 注意事項
- 個人のプライバシーに関わる内容は特に慎重に取り扱うこと
- 感情的な表現は避け、事実のみを記録すること
- 重要な合意事項は別途文書化すること
- 記録は定期的にバックアップを取ること

## 保管期間
- 全ての記録は手続き完了後も5年間保管
- 特に重要な合意事項に関する記録は永年保管
