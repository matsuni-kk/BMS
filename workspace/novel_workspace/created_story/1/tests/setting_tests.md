# 小説設定テストケース

## テスト実施ガイドライン
### テスト実施手順
1. 各テストケースの前提条件を確認
2. テストステップに従って検証を実施
3. 結果を記録し、期待結果との差異を分析
4. 必要に応じて修正案を作成

### 判定基準
- 合格：すべての評価項目が基準値以上
- 条件付き合格：重要項目が基準値以上で、他が許容範囲内
- 不合格：重要項目が基準値未満
